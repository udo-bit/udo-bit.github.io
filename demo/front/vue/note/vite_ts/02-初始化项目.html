02-初始化项目

GXLLess than 1 minute
On This Page
  • 目标
    • 初始化项目

目标

通过pnpm先初始化一个最简单的vite+vue3的项目。并完成依赖的安装。

初始化项目

  • 使用pnpm create vite命令去创建一个基础的vue项目模板
  • 项目名称 naive-admin-pro
  • 选择框架 vue
  • 选择语言 typescript
  • 移动到项目目录，安装依赖
  • cd naive-admin-pro
  • 执行 pnpm i 安装依赖
  • 然后再执行 pnpm dev 启动项目
